Key Flows

How it all connects.

Three core journeys — authentication, job booking, and KYC — each designed to be frictionless.

01
🔐 Auth Flow
  • Animated splash screen on launch
  • Login or Register with credentials
  • Landing Page checks KYC gate
  • Verified → Dashboard. Not verified → Documents.
02
📋 Job Booking Flow
  • Incoming job appears on Active screen
  • Review details — customer, location, time, price
  • Accept or Reject with a single tap
  • Mark Complete when job is done
03
📄 KYC Verification
  • Upload Aadhaar, PAN, Skill Certificate, Police Verification
  • Status polls every 5 seconds automatically
  • Admin reviews → Verified → Auto-navigate to Dashboard
  • Rejected → Re-upload prompt shown immediately
04
🏗️ API Architecture
  • Single kBaseUrl in api_client.dart controls everything
  • Barrel file api.dart — one import for all screens
  • Dedicated modules: auth, bookings, payments, services, users
  • Change backend in one line. Every screen updates.
